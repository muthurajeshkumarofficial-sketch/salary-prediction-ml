# Salary Prediction using Machine Learning

## Project Overview
This project predicts salary based on job-related features using Machine Learning.

## Dataset Features
- experience_level
- employment_type
- job_title
- company_size
- company_location
- remote_ratio

## ML Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Encoding
4. Model Training
5. Model Evaluation

## Model Used
Random Forest Regressor

## Performance
R2 Score: 0.36
MAE: 33444

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy
